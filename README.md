# Remote Database Reader (Computer A -> Computer B)

This folder includes `remote_connect.py`, a simple script that lets **Computer B** read data from a database that lives on **Computer A**.

## What it does
- Computer A hosts the database.
- Computer B connects to Computer A and reads data.

## How to use
1. Configure the connection settings inside `remote_connect.py`.
2. Run the script on Computer B.

That is it.
